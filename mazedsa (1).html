Each REMOVE is a tree edge. DFS produces a perfect spanning tree — exactly one path between any 2 cells.

Array-based LIFO. Push on move, pop when stuck. Blue cells = stack contents.

Array-based FIFO. Guarantees shortest path. Cyan cells = BFS frontier.
